An Linux Emulation installer for NetBSD and FreeBSD



                                        Installation:
NetBSD:
$ git clone http://github.com/glowiak/linux-installer.git && cd linux-installer && sh linux-installer.sh

FreeBSD:
$ git clone http://github.com/glowiak/linux-installer.git && cd linux-installer && sh linux-installer-freebsd.sh


Installing Dependiences:

NetBSD:
$ su root -c "pkgin install git"

FreeBSD:
$ su root -c "pkg install git"
